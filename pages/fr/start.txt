====== L'Open Source dans la radio amateur ======

Bienvenue sur le wiki Open Source en radioamateurisme ! Cette ressource est dédiée à la fourniture d'informations sur les **les logiciels libres** et du **matériel** ainsi que **projets maison gratuits** pour les radioamateurs. L'idée de ce site web ou wiki est de donner à un (futur) radioamateur une vue d'ensemble de tous les projets open source disponibles. L'objectif est de promouvoir l'utilisation de logiciels et de matériel open source dans le domaine de la radio amateur. En fonction des besoins personnels, il est désormais possible de mettre en place une station de radio amateur dont les principaux composants sont open source.
